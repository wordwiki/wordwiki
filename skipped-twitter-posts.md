# Word-a-day posts that need to be recorded by hand

For ~2 years after the old (Shoebox) dictionary was retired, the word-a-day
kept being posted there. We reloaded those posts into the dictionary so the
word-a-day picker stops re-offering already-posted words. Most matched a word
automatically. These **28** did not, so they need a person to record them.

**What to do:** open the correct word on the live site with the link in the
table and mark it posted there (record its twitter-post - the same thing the old
`\tp` field was). Do it in the **current production system**
(mikmaqonline.org) - those edits are picked up automatically the next time we
reload, and this list shrinks until it is empty before the switch to the new
version.

The **post** column is the original entry from the old system (the `#NNNNN` is
the running post number, then the date it was posted).
